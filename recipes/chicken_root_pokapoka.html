鶏肉と根菜のぽかぽか煮込み

材料（1人前）

  • 鶏もも肉 … 100g（一口大に切る）
  • ごぼう … 1/4本（斜め薄切り）
  • にんじん … 1/4本（乱切り）
  • 大根 … 3cm（いちょう切り）
  • 生姜（薄切り） … 2枚
  • だし汁 … 300ml
  • 醤油 … 小さじ1
  • みりん … 小さじ1
  • 塩 … 少々

作り方

  1. 鍋にだし汁を入れ、鶏肉と生姜を加えて中火で煮る。
  2. アクを取り、ごぼう・にんじん・大根を加えて10〜15分煮込む。
  3. 醤油・みりん・塩で味を調える。
  4. 野菜がやわらかくなったら器に盛り付けて完成。

効能メモ

鶏肉は気血を補い、体を温める効果があります。
根菜類は体を内側から温め、生姜と合わせることで冷え改善にも効果的。
